# AYVU NoLag — Changelog

## v0.1.0 — 2026-05-27 (MVP Inicial)

### Adicionado
- Estrutura do projeto WPF .NET 8
- Diagnóstico de ping para endpoints externos
- Cálculo de ping médio, melhor ping, jitter aproximado e packet loss
- Teste de resolução DNS para hosts conhecidos
- Detecção de processos prováveis de jogos ativos
- Boost local com priorização do processo selecionado
- Limpeza leve de memória do app
- Recomendações simples e reversíveis

### Fora do escopo desta versão
- Túnel de rede próprio
- Multipath real
- Driver de rede
- Troca automática de DNS
- Modificação permanente de registro, firewall ou adaptadores
